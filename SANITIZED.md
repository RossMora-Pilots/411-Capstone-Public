# Sanitization Notice

This repository is a public-portfolio version of internal coursework. Identifying details have been redacted so that work product can be shared with prospective employers without exposing third parties.

## What was redacted

- Real client / industry-partner names → "Industry Partner"
- Real instructor names → "Course Instructor"
- Real industry mentor names → "Industry Mentor"
- Group-project teammates' names (in Policies-Compliance) → "Teammate A/B/C"
- External client domain → `example.com`
- Raw meeting transcripts (where present) — replaced with synthesised meeting summaries

## What is preserved

- Technical work, methodology, deliverables, and architectural decisions
- The user's own contributions, role, and reflections
- Course code (CSC-XXXX) and academic context
